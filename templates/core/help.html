Schnellstart-Anleitung
1. Erste Schritte

Nach dem Login können Sie das Dashboard nutzen, um Deals zu verwalten und Templates zu erstellen.

2. Deals erstellen
  1. Gehen Sie zu "Deals" im Hauptmenü
  2. Klicken Sie auf "Neuen Deal erstellen"
  3. Füllen Sie alle erforderlichen Felder aus
  4. Wählen Sie ein Template aus (optional)
  5. Speichern Sie den Deal
3. Dateien hochladen
  1. Öffnen Sie einen Deal
  2. Gehen Sie zu "Dateien"
  3. Klicken Sie auf "Datei hochladen"
  4. Wählen Sie eine Datei aus und kategorisieren Sie sie
  5. Speichern Sie die Datei
4. Templates verwalten
  1. Gehen Sie zu "Templates" im Hauptmenü
  2. Erstellen Sie neue Templates oder bearbeiten Sie bestehende
  3. Laden Sie Template-Dateien hoch
  4. Aktivieren Sie Templates für die Verwendung
